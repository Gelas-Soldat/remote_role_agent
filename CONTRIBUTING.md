# Contributing

This is a small local first job search tool. Keep changes simple, readable, and safe.

## Good contributions

1. Better source adapters.
2. Better United States remote filtering.
3. Better dashboard layout.
4. Better performance on low powered machines.
5. Safer handling of local output and secrets.

## Before opening a pull request

Run:

```bash
python -m py_compile app.py job_agent.py
```
